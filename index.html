Rules

Single Player Multiplayer

1.If player reaches 100 points before the roll count hits zero is declared as winner.

1.The player to reach 100 points first is the winner.


2.In the easy level the roll count is 40 and in the hard level the roll count is 25.

2.In each turn player rolls the dice as many times a he wishes.


3.If hold button is selected then the current score is added to the total score and roll count is decremented by one.

3.If hold button is selected then the current score is added to the total score and it becomes the opponent's turn.

4.If player encounters one on the dice roll a pop up is showen and you'll be given the choice to choose between OK and Cancel If you click OK half of your current score is added to the total score.If you click Cancel current score is made 0.

4.If player encounters one on the dice roll a pop up is showen and you'll be given the choice to choose between OK and Cancel If you click OK half of your current score is added to the total score.If you click Cancel current score is made 0 and it becomes the opponents turn.

5.In the easy level the player gets 3 chances to use the popup box and in hard level the player gets 1 chance to use the popup box.

5.The player gets only 1 chance to use the popup box.

6.If you want to start a fresh game select New Game button.

6.If any player wants to start a fresh game select New Game button.
